Week 5 to 6: JavaScript

During these weeks, we delved into the realm of JavaScript, a foundational programming language essential for web development. JavaScript empowers us to craft dynamic content, manage multimedia elements, animate visuals, and much more, thereby enhancing the interactivity and functionality of web pages.

  • Basics: We explored essential principles like variables, data types, functions, and control structures.
  • Functions/Events: Examined how event listeners and functions can be utilized to effectively interact with web pages.
  • DOM Manipulation: Engaged in exercises involving the modification of content, style, and structure within documents using JavaScript's DOM manipulation capabilities.

Week 7: DOM Events and Local Storage

This week revolved around harnessing the power of DOM Events and Local Storage, indispensable tools for imbuing web pages with interactivity and enabling data storage directly within the user's browser.

  • DOM: Explored the utilization of events to activate JavaScript functions, enhancing user interaction and engagement in the process.
  • Storage: Delved into the mechanics of storing and retrieving data locally, facilitating persistent state management across browsing sessions.

Week 8: API from restdb

APIs play a pivotal role in contemporary web development practices. This week, we gained proficiency in leveraging restdb.io to seamlessly integrate external databases into our projects.

  • API Integration: Practiced making RESTful requests to restdb.io for seamless data storage and retrieval.
  • Data Manipulation: Explored techniques for manipulating and presenting data fetched from an API within our web applications.

Week 9: Bootstrap

Week 9 was dedicated to mastering Bootstrap, a robust front-end framework designed to expedite the development of responsive and visually appealing websites.

  • Responsive Design: Became proficient in crafting responsive layouts using Bootstrap's flexible grid system.
  • Bootstrap Components: Leveraged Bootstrap's extensive library of components such as navbars, modals, and alerts to enhance user interface functionality.

Week 12: REST db

Week 12 marked a transition towards comprehending and harnessing REST db, a cloud-based database service employing RESTful APIs for streamlined database operations.

  • Database Integration: Acquired proficiency in executing CRUD operations (Create, Read, Update, Delete) using RESTful APIs.
  • Data Modeling: Explored strategies for designing efficient database schemas to optimize data storage and retrieval processes.

Week 13: Good Practices for Web Design

Week 13 was devoted to instilling best practices in web design, encompassing aspects such as user interface design, accessibility, and performance optimization.

  • User Experience (UX): Emphasized the significance of user-centric design to create intuitive and engaging web experiences.
  • Accessibility: Explored methods for ensuring web content remains accessible to users of all abilities, aligning with WCAG guidelines.
  • Performance Optimization: Learned techniques for enhancing website speed and efficiency to bolster user satisfaction.

Week 14: LOTTIE and Animation

The final week introduced us to LOTTIE, an innovative library enabling the seamless integration of lightweight and scalable animations into web applications. These animations, based on Adobe After Effects, are converted into a JSON format, ensuring high-quality playback across all devices.

  • Integrating LOTTIE Animations: Explored methods for seamlessly incorporating LOTTIE animations into web projects to enhance visual appeal and user engagement.
  • Animation Principles: Delved into the fundamentals of animation, including timing, easing, and narrative, to craft immersive and impactful interactions.
